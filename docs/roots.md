# Roots

Ethra roots are semantic families, not isolated meanings. A root carries a conceptual field that can generate verbs, nouns, adjectives, agents, objects, ritual forms, civic forms, and intimate forms.

## Pattern System

For a two-consonant root such as `R-H`:

```text
rah      action/verb
reh      concept/noun
rih      quality/adjective
rahen    agent
roh      object/presence
ha-ruh   ritual form
rah-da   civic/legal form
mi-rah   intimate/emotional form
```

For a three-consonant root such as `S-N` or future four-consonant roots, vowels are inserted between consonants to avoid clusters.

## Core Root Families

```text
R-H   seeing, revealing, witnessing, recognition, truth, face, presence

P-T   path, journey, destiny, direction

N-M   speech, naming, calling, command
F-R   future, unborn, tomorrow, consequence

```

The full machine-readable root list is in `spec/roots.yaml`.

## Why Roots Matter

Roots make memory visible inside vocabulary. A speaker who hears `rah`, `reh`, `rih`, and `mi-rah` hears a family: seeing, recognition, visible truth, and intimate recognition. This creates conceptual gravity. Words become related across law, love, prayer, and philosophy.

# Ethra Corpus

This folder holds reviewed Ethra usage. The corpus exists to keep vocabulary grounded in sentences, dialogues, civic forms, ritual speech, technical writing, literature, and learner material.

Track definitions and quality gates are in `spec/corpus-plan.yaml`.

## Tracks

```text
daily-dialogues/
civic-law/
ritual-vow/
technical-software/
literary-poetic/
learner-graded/
```

Each corpus item should include an English source, Ethra text, literal translation, notes, domain tags, and any terms introduced.
